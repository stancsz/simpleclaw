# SimpleClaw Skills
This directory contains agent skills compatible with SimpleClaw.
Skills are markdown files that provide instructions to the AI on how to use specific tools or perform complex workflows.

## Included Skills
1. [browser.md](./browser.md) - Web automation via `agent-browser`.
2. [shell_master.md](./shell_master.md) - System operations.

## Adding More Skills
You can add any SKILL.md from [Anthropic's skills repository](https://github.com/anthropics/skills) here.
The agent will automatically be aware of these instructions if they are included in the system prompt.
