# SimpleClaw Electron Assets

Place application icons here:

- `icon.png` (512x512) - Main application icon
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon

For production, generate proper icons using tools like:
- `png2icons` for macOS .icns files
- `icotool` or online converters for Windows .ico files
- ImageMagick for resizing: `convert icon.png -resize 256x256 icon-256.png`
